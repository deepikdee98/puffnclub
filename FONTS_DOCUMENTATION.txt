üìö FONT FAMILIES
================================================================================

1. INSTRUMENT SANS (Default/Body Font) ‚≠ê PRIMARY
   - Used for: EVERYTHING by default (headings, body text, buttons, labels, etc.)
   - Weights: 400 (Regular), 500 (Medium), 600 (Semi-Bold), 700 (Bold)
   - CSS Variable: var(--font-instrument-sans)
   - Supports: Regular and Italic styles
   - Default Font: All h1-h6, paragraphs, and all text elements

2. BEBAS NEUE (Display Font) - USE WITH CLASS ONLY
   - Used for: Special headings/text ONLY WHERE SPECIFIED in Figma
   - Weight: 400 (Regular only)
   - Style: Uppercase, bold display font
   - CSS Variable: var(--font-bebas-neue)
   - Characteristics: Letter-spacing: 1px, Text-transform: uppercase
   - ‚ö†Ô∏è IMPORTANT: Apply using className only, NOT automatic


üìã FONT WEIGHT REFERENCE
================================================================================

Instrument Sans Weights:

‚úÖ BEST PRACTICES & GUIDELINES
================================================================================

‚ö†Ô∏è NEW APPROACH (Class-Based System):

1. DEFAULT FONT (Instrument Sans):
   - Everything uses Instrument Sans by default
   - No className needed for normal text
   - Includes: All headings (h1-h6), body text, forms, buttons, etc.

2. WHEN TO USE BEBAS NEUE (className="font-bebas"):
   - ‚úÖ Main hero headings on landing pages
   - ‚úÖ Large section titles (only if specified in Figma)
   - ‚úÖ Special callout text or banners
   - ‚úÖ Promotional text/badges
   - ‚ùå DO NOT use for: Product names, card titles, regular headings
   - ‚ùå ONLY apply where Figma design specifically shows it

3. HOW TO APPLY BEBAS NEUE:
   Step 1: Check Figma design
   Step 2: If Bebas Neue is specified, add className
   Step 3: Use any of these: "font-bebas", "heading-bebas", or "bebas"

   Example:
   <h1 className="font-bebas">WELCOME TO PUFFN CLUB</h1>

4. Font Weight Guidelines (Instrument Sans):
   - 400 (Regular): Default body text
   - 500 (Medium): Emphasis, subtle highlighting
   - 600 (Semi-Bold): Subheadings, important labels
   - 700 (Bold): Strong emphasis, key information

5. Color Usage:
   - Use CSS variables for consistency
   - Prefer Bootstrap classes for common colors
   - Use hex codes only when necessary


üîß CUSTOMIZATION
================================================================================

To add new font weights for Instrument Sans:
Edit /frontend/src/app/layout.tsx

const instrumentSans = Instrument_Sans({
  subsets: ["latin"],
  weight: ['400', '500', '600', '700', '800'], // Add 800 if needed
  variable: '--font-instrument-sans',
  display: 'swap',
});

To add new colors:
Edit /frontend/src/styles/globals.scss

:root {
  --custom-color: #123456;
}
